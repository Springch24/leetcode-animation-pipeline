# 自学算法 — 完整产品方案

---

## 一、核心闭环

```
你产内容 → 观众看完 → 扫码进小程序 → 刷题打卡 → 数据反馈 → 优化内容
    ↑                                                              |
    └────────────────── AI 自动生产 ←──────────────────────────────┘
```

三条线并行：
- **内容线**：你（+ AI）生产视频，B站/抖音分发
- **产品线**：小程序承接流量，提供刷题 + AI 学习工具
- **数据线**：cheat-on-content 分析数据，反馈指导下一步内容策略

---

## 二、小程序功能模块

### MVP（第一版，2周搞定）

| 模块 | 功能 | 复用现有代码 |
|------|------|------------|
| 题目列表 | 跟视频一一对应的 LeetCode 题目，点进去有题解+视频链接 | 新建 |
| 刷题打卡 | 今天刷了几道？打卡记录，连续天数 | DLM 的 records 表复用 |
| AI 代码评估 | 用户贴代码，AI 评分 + 给改进建议 | Gemini API（已有） |
| 个人统计 | 刷题日历热力图、总题数、连续天数 | DLM 的 Stats 页面复用 |
| 视频入口 | 每道题关联你的B站视频链接 | 新建 |

### V2（涨粉后）

| 模块 | 功能 |
|------|------|
| AI 模拟面试 | 选一道题，AI 扮演面试官，实时对话 |
| 排行榜 | 周榜/月榜，刷题数量排名 |
| 每日一题推送 | 公众号模板消息 + 小程序通知 |
| 知识星球/付费群 | 深度内容变现 |

---

## 三、技术方案

```
前端：React + TailwindCSS（DLM 现有技术栈）
后端：Express + SQLite（DLM 现有技术栈）
AI：Gemini API（评估代码 + 生成题解）
打包：微信小程序用 Taro/uni-app，抖音小程序类似
```

跟 DLM 的区别：
- DLM 是全栈 Web + Capacitor 打包 App
- 自学算法需要微信小程序原生适配（更轻量）
- 用 Taro 框架，一份代码编译微信 + 抖音小程序

## 四、视频 → 小程序导流设计

每条视频结尾固定 3 秒引导画面：

```
┌─────────────────────────┐
│                         │
│    📱 扫码做题打卡       │
│                         │
│   [小程序码]            │
│                         │
│   每道题都有 AI 评估     │
│                         │
└─────────────────────────┘
```

- B站视频：结尾 + 评论区置顶放小程序码
- 抖音视频：简介放「微信搜『自学算法』小程序」

## 五、内容生产节奏

| 频率 | 平台 | 内容 |
|------|------|------|
| 每天 1 条 | B站 | 完整动画教程（横版） |
| 每天 1 条 | 抖音 | 30s 精华版（竖版） |
| 每周 1 条 | B站 | 人设/幕后/复盘视频 |
| 每天 | 小程序 | 跟视频同步上线对应题目 |

## 六、变现路径（按优先级）

1. **小程序广告**（最早变现）——微信小程序激励视频广告，用户看广告解锁 AI 模拟面试
2. **B站知识付费**（中期）——打包 LeetCode Hot 100 全部视频 + PDF 题解，卖 9.9
3. **付费社群**（后期）——知识星球/微信群，每日一题 + AI 批改 + 模拟面试
4. **企业内推**（长期）——粉丝多了之后，接面试辅导和内推合作

## 八、你现在可以立刻做的事
