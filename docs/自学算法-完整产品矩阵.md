## 二、小程序功能矩阵

### Phase 1：刷题工具（MVP）
| 模块 | 功能 |
|------|------|
| 视频题解 | 每道题对应你的视频 + 文字题解 + 代码 |
| 刷题打卡 | 每日打卡，连续天数，热力图 |
| AI 代码批改 | 用户贴代码 → Gemini 评分 + 建议 |
| 题目收藏夹 | 按公司/知识点分类收藏 |

### Phase 2：AI 学习助手
| 模块 | 功能 |
|------|------|
| AI 解题助手 | 卡住了？描述思路，AI 给提示但不给答案 |
| AI 复杂度分析 | 贴代码 → 自动分析时间/空间复杂度 |
| 错题本 | 自动收集你做错的题，定期提醒复习 |
| 学习路线推荐 | 根据你的水平，AI 推荐刷题顺序 |

### Phase 3：AI 面试
| 模块 | 功能 |
|------|------|
| AI 模拟面试官 | 选公司 + 难度 → AI 出题 → 你口述/打字 → AI 追问 → 打分 |
| 面试录音分析 | 录下真实面试，AI 分析你的表达、逻辑、技术深度 |
| 白板编程模拟 | 在画布上手写代码，AI 实时提示 |
| 行为面试训练 | AI 问「你最大的缺点是什么」之类的问题，评估回答 |
| 面试复盘报告 | 每次模拟面试后生成报告：哪里好、哪里改进 |

### Phase 4：社区 + 变现
| 模块 | 功能 |
|------|------|
| 面经广场 | 用户分享真实面经（匿名），AI 提取关键信息 |
| 排行榜 | 刷题榜 / 面试分榜 / 连续打卡榜 |
| 付费题库 | 大厂最新真题（按月更新，付费解锁） |
| 1v1 模拟面试 | 预约你的时间，真人模拟面试（高价） |
| 内推广场 | 大厂员工发内推码，你审核后展示 |
