## 接线方案

```
USB麦克风 ──┐
USB摄像头 ──┤
USB喇叭 ────┼── 树莓派 ── WiFi ── Hermes API
OLED(I2C) ──┤
红外发射 ───┘   (GPIO 17)

OLED: SDA→GPIO2, SCL→GPIO3, VCC→3.3V, GND→GND
红外发射: DAT→GPIO17, VCC→5V, GND→GND
红外接收: DAT→GPIO18, VCC→3.3V, GND→GND
```
