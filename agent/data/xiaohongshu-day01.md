# Agent 到底是个啥？四句话讲清楚 🤖

> 不是 ChatGPT，不是自动化脚本。6000 人看过了，114 人收藏了这份笔记。

---

## 一句话定义

**Agent = 传感器感知环境 → 自主决策 → 执行器行动**

四个词：**感知 → 决策 → 行动 → 目标**

---

## 传统 Agent vs LLM Agent

|  | 传统 Agent | LLM Agent |
|---|---|---|
| 核心 | 工程师写规则 | 大模型驱动 |
| 知识来源 | 人工编写 | 海量预训练 |
| 行为特点 | 确定性强，边界清晰 | 灵活通用，能处理模糊指令 |
| 举例 | 恒温器 (if 温度>设定 → 制冷) | "帮我规划一趟厦门旅行" |

---

## 四类 Agent 进化之路

1. **简单反射型** — 纯条件-动作，没记忆。像恒温器
2. **基于模型型** — 有内部世界模型。像自动驾驶穿过隧道还能"记住"前面有车
3. **基于目标型** — 不只反应，会规划。像导航选路线
4. **基于效用型** — 不只达成目标，还要"最优"。像交易算法

---

## 一句话记住

> **传统 Agent 是工程师写规则的自动化工具，LLM Agent 是能理解模糊指令、自主规划行动的通用智能体。**
