# 人设视频拍摄方案：「AI 自动做 LeetCode 动画」

---

## 一、核心定位

**一句话钩子**：我把 LeetCode Hot 100 全部做成了动画视频 —— 但不是我做的，是我写的代码做的。

**人设关键词**：算法工程师、自动化、AI工具流、效率狂魔

**目标观众**：程序员、技术爱好者、想学AI的人

---

## 二、视频结构

### 抖音版（竖版 35-45 秒）

| 秒 | 画面 | 台词 |
|----|------|------|
| 0-3 | 你对着镜头，自然说话 | 「你还在手写 LeetCode 题解吗？我已经让 AI 帮我做了 100 道。」 |
| 3-8 | 录屏：终端跑 `./build.sh`，HyperFrames 开始渲染 | 「看，我写了一套自动化流水线。输入一道题的 JSON，自动生成动画视频，连配音都是 AI。」 |
| 8-15 | 穿插成品画面：两数之和的动画片段（哈希表查找那段） | 「哈希表怎么查？动画一目了然。代码逐行高亮，数据结构可视化。」 |
| 15-22 | 录屏：封面自动生成过程 | 「连封面都是自动生成的，统一视觉风格。」 |
| 22-30 | 展示三个视频排成一排的文件夹截图 | 「一天做一道，现在已经三道了。目标是做完 LeetCode Hot 100。」 |
| 30-35 | 回到镜头前 | 「完整视频在 B 站搜『自学算法』。这套自动化工具，你想学的话评论区告诉我。」 |

### B站版（横版 2-3 分钟）

| 秒 | 画面 | 台词 |
|----|------|------|
| 0-10 | 你对着镜头，坐姿自然 | 「大家好，我是一个算法工程师。最近我在做一个实验——用 AI 和自动化工具，每天生产一条 LeetCode 动画讲解视频。」 |
| 10-15 | **画面转场**：动画标题卡「AI 自动化内容生产实验」 | （背景音乐轻快） |
| 15-30 | 录屏：展示目录结构，介绍工具链 | 「这是我的内容工厂。HyperFrames 负责渲染动画，edge-tts 生成中文配音，cheat-on-content 做数据复盘。全套工具链打通。」 |
| 30-50 | **重点片段**：录屏跑一条完整流水线（10秒加速） | 「一条视频的生产流程是这样的：我先准备题目的 JSON 数据，脚本自动生成 HTML 动画模板，GSAP 控制每一帧的动画时间线，然后 CLI 渲染成 MP4。」 |
| 50-75 | 穿插三个成品的精华片段（两数之和、两数相加、最长连续序列） | 「效果怎么样？给你们看看成品。」 |
| 75-95 | 封面 + 数据看板 | 「封面也是自动生成的。每道题风格统一，做一百道就有一百张高质量封面。发布后还能用数据工具追踪表现。」 |
| 95-115 | 你对着镜头 | 「我做这件事不是因为闲，而是因为我相信——未来的内容生产一定是 AI 驱动的。我现在做的事，就是把这个未来提前拉到今天。」 |
| 115-130 | 录屏：GitHub 或者文件夹截图 | 「这套流程完全可复制。如果你想搭建自己的自动化内容流水线——不管是做技术教程、知识分享、还是个人 IP——」 |
| 130-140 | 回到镜头前，真诚 | 「关注我，我会把这个系列的代码和工具开源出来。我们一起用 AI 重新定义内容创作。」 |
| 140-150 | 结尾卡：B站搜索「自学算法」+ 评论区引导 | 「评论区告诉我，你最想看哪道题的动画？」 |

## 三、拍摄清单

### 器材
| 项目 | 方案 |
|------|------|
| 相机 | iPhone 后置摄像头即可（4K 30fps） |
| 收音 | **重点**：用你日常用的耳机带麦就行，但要靠近嘴 |
| 灯光 | 白天靠窗自然光，晚上用台灯从侧前方打光 |
| 背景 | 干净的书桌/书架即可，不要太乱，不要白墙 |

### 机位
- 横版（B站）：手机横放，你坐在桌子前，半身入镜，眼睛看镜头
- 竖版（抖音）：手机竖放，同样构图，稍微更近一些

### 录屏
| 内容 | 工具 |
|------|------|
| 终端跑命令 | macOS 自带 `Cmd+Shift+5` 录屏，区域选终端窗口 |
| 动画成品 | 直接从成品视频里截取片段 |
| 文件夹/VS Code | 录屏软件截取 |

### 剪辑工具
- 用 **剪映专业版**（免费，Mac 可用）
- 录屏 + 你对着镜头的画面 + 成品视频片段 = 拼在一起就行

## 四、拍摄话术要点

**不要念稿**。记住要点，对着镜头自然说：

1. **开场钩子**：制造反差（「不是我做的，是我写的代码做的」）
2. **展示实力**：快速炫技（工具链、流程、成品）
3. **拉共鸣**：「你也可以」
4. **留钩子**：引导关注/评论

**语气**：自然、分享、不装逼。不要像讲课，要像和朋友分享一个有意思的事。

**眼神**：看镜头，不要看屏幕。想象镜头是你的一个程序员朋友。

---

## 五、剪辑节奏建议

- 画面切换节奏要快（2-4 秒一个镜头）
- 你说话的镜头 + 录屏/成品画面穿插
- 不要让你一个人对着镜头说超过 8 秒
- 背景音乐：剪映自带的轻科技感 BGM，音量降到 15%

## 七、拍摄流程（15 分钟搞定）

1. **提前准备**（5min）：打开终端、准备好要录屏的窗口、调好手机位置
2. **录 A-roll**（5min）：你对着镜头说话的部分，可以分段录，后期拼
3. **录屏**（3min）：跑命令、展示文件、展示成品，每个 10 秒左右
4. **导入剪辑**（在剪映里拼）：A-roll + 录屏 + 成品片段交叉排
