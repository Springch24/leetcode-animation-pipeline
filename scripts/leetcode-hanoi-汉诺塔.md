# 汉诺塔 — 递归经典（粉丝点题）

## 标题
面试官让你手写汉诺塔，90%的人说简单递归三行。面试官说——用迭代模拟递归栈。

## 开场钩子（0-6s）
三根柱子，n个盘子，大的不能在小的上面。这就是汉诺塔——递归入门题，但面试官要你用迭代写。

## 题目描述（6-14s）
汉诺塔问题：n个大小不同的盘子从A柱移到C柱，借助B柱。每次只能移动一个盘子，大盘不能放在小盘上。求最少步数并输出每一步。

## 解法核心（14-55s）
递归思路：把 n 个盘子从 A 移到 C，等价于：
1. 把上面 n-1 个从 A 移到 B（借助 C）
2. 把最大盘子从 A 移到 C
3. 把 n-1 个从 B 移到 C（借助 A）

最少步数 = 2^n - 1。3个盘子要7步。

**迭代版**：用栈模拟递归调用。每次入栈记录 (n, from, to, aux) 四个参数。出栈时如果 n=1 直接移动，否则拆成三个子问题入栈。
