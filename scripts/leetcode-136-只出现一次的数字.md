# LeetCode 136 — 只出现一次的数字

## 标题
面试官让你找出数组中只出现一次的数字，90%的人用哈希表。面试官说——不用额外空间能做到吗？

## 开场钩子（0-6s）
一个数组里除了一个数字只出现一次，其他都出现两次。90%的人秒写哈希表。面试官说——空间O(1)呢？答案：异或。

## 题目描述（6-14s）
LeetCode 136 只出现一次的数字。给定一个非空整数数组，除了某个元素只出现一次，其余每个元素均出现两次。找出那个只出现一次的元素。要求线性时间，不用额外空间。

## 解法核心（14-48s）
异或运算三条性质：
1. a ⊕ a = 0（相同数字异或得0）
2. a ⊕ 0 = a（任何数异或0还是自己）
3. 交换律和结合律（顺序无关）

所以把数组所有数字异或起来：成对的数字两两消成0，最后剩下那个单独的。

代码只有一行：`reduce(xor, nums)`

动画演示 [4,1,2,1,2]：4⊕1=5 → 5⊕2=7 → 7⊕1=6 → 6⊕2=4

## 复杂度（48-56s）
- 时间 O(n)，一遍遍历
- 空间 O(1)，只用一个变量

## 金句（56-64s）
**"相同数字异或得0，全场异或剩下那个单独的。"**
