# LeetCode 121 — 买卖股票的最佳时机

## 标题
面试官让你算股票最大利润，90%的人上来就双重循环。面试官说——O(n) 一遍过能做到吗？

## 开场钩子（0-6s）
给你一个数组表示每天股价，只买卖一次，求最大利润。90%的人写暴力双重循环 O(n²)。面试官说——你的时间复杂度太高了。

## 题目描述（6-14s）
LeetCode 121 买卖股票的最佳时机。给定一个数组 prices，prices[i] 是第 i 天股票价格。选择某一天买入，未来的某一天卖出，求能获得的最大利润。如果不能盈利返回 0。

## 解法核心（14-50s）
关键：一遍扫描，维护两个变量。

- `minPrice` = 到当前位置的最低买入价
- `maxProfit` = 到当前位置的最大利润

遍历每一天：如果当天价格比 `minPrice` 低，更新最低价。否则算当天利润（价格 - minPrice），如果比 `maxProfit` 大就更新。

例 [7,1,5,3,6,4]：
- day0: minPrice=7, maxProfit=0
- day1: 1<7 → minPrice=1
- day2: 5-1=4 → maxProfit=4
- day3: 3-1=2 → 4 更大，不变
- day4: 6-1=5 → maxProfit=5
- day5: 4-1=3 → 5 更大，不变
答案: 5

## 复杂度（50-58s）
- 时间 O(n)，一遍扫描
- 空间 O(1)，两个变量

## 金句（58-66s）
**"记录最低买入价，每天算一次利润，取最大。"**
