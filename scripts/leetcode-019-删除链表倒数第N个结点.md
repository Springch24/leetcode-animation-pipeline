## 旁白稿
面试官让你删除链表的倒数第N个节点。90%的人第一反应：先遍历一遍数长度，再遍历一遍删。面试官说——能一遍过吗？

LeetCode 19。给你一个链表，删除倒数第N个节点。比如 1→2→3→4→5，N=2，删掉倒数第2个节点 4，变成 1→2→3→5。

怎么做？快慢指针。快指针先走 N 步。然后快慢一起走，快指针走到末尾时，慢指针刚好在要删的节点前面。直接跳过即可。

看动画。快指针先走 2 步，指向 3。然后快慢一起走——快指到 5，慢指到 3。慢指针在 4 前面，跳过 4。搞定。

看代码。快指针先走 N 步，然后快慢同步前进，最后跳过慢指针的下一个节点。注意边界：删的是头节点怎么办？用虚拟头节点。

时间复杂度 O(n)，一遍扫描。空间复杂度 O(1)，只用了两个指针。

一句话记住：快指针先走 N 步，然后一起走，快到底时慢的下一个就是要删的。
