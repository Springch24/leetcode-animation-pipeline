## 旁白稿

面试官让你删除排序链表里的重复元素，90%的人第一反应：遍历存数组，去重再建链表。面试官说——原地 O(1) 空间能做到吗？

LeetCode 83。给你一个排序链表，比如 1→1→2→3→3，删掉重复节点后变成 1→2→3。

怎么做？一个指针就够了。cur 从头开始，比较 cur 和 cur.next 的值。如果相等，cur.next 跳到下下个节点，跳过重复。不相等就 cur 往前走。非常简单。

看动画。cur 指向第一个 1，cur.next 也是 1，相等——跳过。cur 指向 1，cur.next 是 2，不相等——cur 前进。cur 指向 2，cur.next 是 3，不相等——前进。cur 指向 3，cur.next 也是 3，相等——跳过。cur 走到末尾，结束。链表变成 1→2→3。

看代码。while cur 和 cur.next 都存在：值相等就跳过，不相等就前进。最后返回 head。

时间复杂度 O(n)，每个节点最多访问一次。空间复杂度 O(1)，只用一个指针。

一句话记住：排序链表去重，值相等跳过，不相等前进。
