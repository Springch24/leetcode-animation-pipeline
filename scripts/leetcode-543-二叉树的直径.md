## 旁白稿
面试官让你求二叉树的直径。90%的人第一反应：左子树深度加右子树深度。面试官说——那最长路径不经过根怎么办？

LeetCode 543。二叉树的直径是任意两个节点之间的最长路径。比如这棵树，直径是 4→2→1→3，长度为 3。注意：直径不一定经过根节点。

怎么做？后序遍历。每个节点计算：左子树深度加右子树深度，用这个值更新全局最大直径。然后返回 max(左深度, 右深度) + 1 给父节点。

看动画。从叶子开始。4 的左右都是 0，直径候选 = 0，返回深度 1。5 同样返回深度 1。回到 2，左深度 1，右深度 1，直径候选 = 2，更新最大直径为 2，返回深度 2。3 的左右深度为 0，直径候选 0，返回深度 1。回到 1，左深度 2，右深度 1，直径候选 = 3，更新最大直径为 3，返回深度 3。答案：3。

代码。全局变量存最大直径。递归函数返回深度。每个节点：左深度 + 右深度 更新 max，返回 max(左,右) + 1。

时间复杂度 O(n)，每个节点遍历一次。空间复杂度 O(h)，递归栈。

一句话记住：深度归父节点，直径留给自己。
