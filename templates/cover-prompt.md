# 公众号封面 Prompt 模板

## 基础约束（每次必须包含）

```
公众号封面，16:9横版。所有文字严格集中在画面正中央约60%区域内，
四周纯深色科技蓝背景+柔和光效（四周会被裁切）。
机器之心科技媒体风格，干净现代，光影有质感，4K画质。
```

## 品牌元素（固定）

- 顶部小字：`橙序员·AI科技媒体`
- 底部：橙色分割线 + `VOL.XX`
- 配色：橙色 #FCA311 + 深蓝背景

## 标题区域（中央，可变）

- 大号白色英文标题
- 中等白色中文标题
- 小字副标题
