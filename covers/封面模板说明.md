# LeetCode 视频封面模板

## 布局结构 (1080×1920)

```
    ─── 顶部装饰线 ───
         LeetCode
           543           ← 题号，200px 橙色大字
    ─── 分割线 ───
      二叉树的直径        ← 中文题名，80px 白色
   Diameter of Binary Tree ← 英文，36px 灰色
    ─── 标签行 ───
    简单  二叉树  DFS     ← 难度+标签，圆角边框
    ─── 间距 ───
      后序遍历             ← 知识点1
      深度归父节点          ← 知识点2
      直径留给自己          ← 金句，橙色大字
    ─── 底部装饰线 ───
   橙序员 · LeetCode 动画教程
```

## 颜色规范
- 背景: #060b14
- 橙色主色: #FCA311
- 白色: #DCE4F0
- 灰色: #4A6080
- 绿色(简单): #4ADE80
- 紫色(DFS/递归): #A78BFA

## 字体
- macOS STHeiti Medium
- 题号: 200px
- 标题: 80px
- 标签/英文: 36px
- 知识点: 22-36px
